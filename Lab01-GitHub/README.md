 Lab 1: Git & GitHub for Machine Learning Projects

 -Lab Objectives
The goal of this lab was to establish a professional development environment using Git and GitHub, which is essential for tracking experiments and collaborating on machine learning models.

-Key Learnings & Skills Acquired

 1.Version Control Fundamentals
Repository Management: Understanding how to initialize and structure a repository specifically for ML projects.
Commit Best Practices: Learning to write descriptive commit messages that reflect the evolution of the project.

 2.Collaboration & Documentation
README Importance:Creating comprehensive documentation to explain the project's purpose and workflow.
Markdown Proficiency: Using Markdown to create organized, readable documentation including tables, lists, and formatting.

This lab laid the foundation for professional workflow quality that will be maintained throughout the semester
